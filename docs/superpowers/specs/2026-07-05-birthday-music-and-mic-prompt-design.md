# 麦克风提示与生日音乐设计

## 目标

在不修改两张生日主视觉图片的前提下，降低麦克风提示对画面的遮挡，并把蜡烛熄灭后的三声提示音替换为完整的生日旋律。

## 提示状态

- 未监听：显示 `轻触开启麦克风`。
- 已监听：只显示 `对着手机底部吹气`。
- 两种提示使用同一位置，整体比当前版本下移约 30–40px。
- 提示底色改为半透明粉白色，不显示边框。
- 删除监听状态出现的黄色矩形边框。

## 音乐

- 蜡烛熄灭后播放一次约 12–15 秒的慢拍爵士钢琴版《Happy Birthday》主旋律。
- 编配使用柔和电钢琴音色、低音与七和弦，不循环。
- 音乐通过 Web Audio API 实时生成，不加载外部录音。
- 用户轻触开启麦克风时预先激活音频上下文，确保吹气触发后仍能播放。
- 右上角声音开关关闭时不播放；播放过程中关闭时立即停止。

## 验收

- 390×844 页面中提示不遮挡蛋糕或手写装饰字。
- 监听状态没有黄色边框。
- 点击蜡烛和吹气触发均调用同一段生日音乐。
- 自动测试覆盖提示状态、音频预激活、静音和音乐时长。
